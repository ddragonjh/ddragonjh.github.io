Hugging Face: 401 unauthorized for gated model

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

Medium urgencyModel access fix

Fast answer

1

Open the model page while signed in and request or accept access.

2

Authenticate the runtime with a read token from the approved account.

3

Verify the exact model id and retry after access is granted.

Confirm the match

Symptoms

  • Downloading a model returns 401, 403, or repository not found.
  • Public models work but one gated model fails.
  • The model page is visible in a browser but code cannot fetch its files.

Why it happens

Likely causes

  • The account has not accepted the model's access conditions.
  • The runtime is not authenticated with a token that can read the repository.
  • The model id, revision, or organization name is incorrect.

Diagnostics

Commands to check

hf auth whoamihf auth loginpython -m pip show huggingface_hub

Safety note

Do not place a Hugging Face token in source control or client-side browser code.
