npm: Missing script: dev

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

Medium urgencyProject script setup fix

Fast answer

1

List the scripts actually defined in the current package.

2

Confirm the working directory and repository branch.

3

Run the documented existing script or restore the intended script from the project source.

Confirm the match

Symptoms

  • npm run dev exits because no dev script is defined.
  • The repository documentation expects a command that is absent locally.
  • Another workspace package may contain the script while the current package does not.

Why it happens

Likely causes

  • The command is running from the wrong package or monorepo directory.
  • The project uses a different script name such as start, serve, or preview.
  • package.json is outdated, incomplete, or from the wrong branch.

Diagnostics

Commands to check

npm runnpm prefixgit status --short

Safety note

Do not guess a framework command until you identify the package and scripts the repository actually uses.
