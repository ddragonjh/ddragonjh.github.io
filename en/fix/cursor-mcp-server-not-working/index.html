Cursor: MCP server not working

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

High urgencyDeveloper setup fix

Fast answer

1

Validate the JSON and replace relative paths with absolute paths.

2

Run the MCP command manually from a clean terminal.

3

Restart the editor after editing the config file.

Confirm the match

Symptoms

  • The MCP tool does not appear in the editor.
  • The server starts in a terminal but the editor cannot call it.
  • The same command works on one machine but not another.

Why it happens

Likely causes

  • The MCP config file is in the wrong location or has invalid JSON.
  • The command needs an absolute path or a different runtime.
  • Environment variables are missing when the editor launches the process.

Commands to check
