Docker: Cannot connect to the Docker daemon

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

High urgencyContainer runtime fix

Fast answer

1

Start Docker Desktop or the operating-system Docker service.

2

Check the active Docker context and remove an unintended DOCKER_HOST value.

3

Fix user access to the socket, then sign out and back in if group membership changed.

Confirm the match

Symptoms

  • Every Docker command fails before it reaches a container.
  • The error references the Docker daemon, socket, or named pipe.
  • Docker worked before a reboot or desktop-app update.

Why it happens

Likely causes

  • Docker Desktop or the Docker service is stopped.
  • The current user cannot access the Docker socket.
  • DOCKER_HOST points to an unavailable remote daemon.

Diagnostics

Commands to check

docker infodocker context showprintenv DOCKER_HOST

Safety note

Do not make the Docker socket publicly writable; it grants host-level control.
