Claude Code: MCP server disconnected

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

High urgencyMCP connection recovery

Fast answer

1

Run the configured command manually and resolve its first startup error.

2

Replace relative executable paths with absolute paths.

3

Restart Claude Code after correcting the MCP configuration and environment variables.

Confirm the match

Symptoms

  • MCP tools disappear or show a disconnected status.
  • The server process exits as soon as Claude Code starts it.
  • The server works in a terminal but not from the coding agent.

Why it happens

Likely causes

  • The configured command or working directory is wrong.
  • Required environment variables are unavailable to the launched process.
  • The server writes invalid protocol output to standard output and closes the connection.

Diagnostics

Commands to check

claude mcp listnode --versionpython --version

Safety note

Keep protocol messages on stdout and send diagnostic logs to stderr.
