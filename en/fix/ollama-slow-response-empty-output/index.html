Ollama: slow response or empty output

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

Medium urgencyLocal AI performance fix

Fast answer

1

Test with a smaller quantized model.

2

Check whether the GPU is being used.

3

Reduce context size and stop other local model processes.

Confirm the match

Symptoms

  • A local model answers very slowly or not at all.
  • CPU usage is high while GPU usage is low.
  • Smaller models respond but larger models stall.

Why it happens

Likely causes

  • The model is too large for available RAM or VRAM.
  • GPU acceleration is not active.
  • Context length or concurrent requests overload the machine.

Diagnostics

Commands to check

ollama listollama psnvidia-smi

Safety note

Start with a smaller model before tuning runtime settings.
