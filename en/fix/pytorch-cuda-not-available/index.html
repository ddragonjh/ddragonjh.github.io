PyTorch: CUDA is not available

Use this checklist in order. Stop when the error is resolved, and do not apply unrelated system changes from later steps.

High urgencyGPU environment fix

Fast answer

1

Confirm the operating system sees the GPU with nvidia-smi.

2

Check the active Python environment and PyTorch build information.

3

Install the PyTorch build recommended for your platform, then restart the process.

Confirm the match

Symptoms

  • torch.cuda.is_available() returns false.
  • Training runs on CPU even though an NVIDIA GPU is installed.
  • The driver sees the GPU but the Python environment does not use it.

Why it happens

Likely causes

  • A CPU-only PyTorch build is installed.
  • The NVIDIA driver is missing, incompatible, or unavailable inside the container.
  • The notebook or shell is using a different Python environment.

Diagnostics

Commands to check

nvidia-smipython -c "import torch; print(torch.__version__, torch.version.cuda, torch.cuda.is_available())"python -m pip show torch

Safety note

Match the installation command to the current PyTorch selector instead of guessing a CUDA wheel.
