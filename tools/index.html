Designed as a workflow

Search less, preserve more evidence

Start with the exact error matcher when the message is recognizable. Use stack trace triage or dependency drift when the first answer is not enough, then carry a sanitized report into Debug Lab for verification and handoff.

The tools produce hypotheses, not guarantees. Their value is the investigation order: isolate the earliest useful failure, compare against a known-good state, and prove the expected outcome after one reversible change.
